# Análise de Vendas

## Objetivo
Analisar dados de vendas para identificar o faturamento total, os produtos mais lucrativos e a evolução das vendas ao longo do tempo.

## Tecnologias utilizadas
- Python
- Pandas
- Plotly

## Etapas da análise
- Carregamento e exploração inicial dos dados
- Limpeza e tratamento dos dados
- Criação de métricas (faturamento)
- Análise exploratória e visualização dos dados
- Interpretação dos resultados

## Principais insights
- Faturamento total no período: R$ 6.960,00
- Produto mais lucrativo: Notebook
- As vendas apresentaram oscilações ao longo dos meses, com um aumento significativo em março de 2024

O projeto segue o ciclo completo de análise de dados (Ask, Prepare, Process, Analyze, Share, Act), conforme o framework do Google Data Analytics.
